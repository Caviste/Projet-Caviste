Projet Caviste 2020

Un projet par Nathan,Radad et Amine pour l'EPFC

Qui nous sommes ?

Nous sommes trois étudiants de l'EPFC en premiére année en web developpeur.

Quel est le but du projet ?

Dans le cadre de notre cour de WSCV, nous devons réaliser un projet de groupe qui s'intitule : PROJET WEB HTML5 – DÉVELOPPEMENT D’UNE SPA

Nous avons dû réaliser le site client Mon cellier.En utilisant une API restful: qui permet a n'importe quel utilisateur de consulter notre catalogue des vins ainsi qu'à des utilisateurs connectés de pouvoir modifier certaines données à l'aide d'un formulaire.

Méthodologie utilisée

Durant ce travail nous avons dû définir les problématique qui sont

  1. Identifier les acteurs et leurs fonctionnalit
  2. Analyser les cas d'utilisation
  3. Identifier les outils nécessaires
  4. Installer et configurer les différents outils de développement
  5. Identifier les contenus gérés par l’application
  6. Identifier les éléments de conception graphique
  7. Commencer à développer

Concrètement que fait l'application Mon cellier?

Dans ce projet, nous gérons la cave à vins d'une entreprise fictive ici Millésime

Comme dit précédement , chaque utilisateur peut consulter le catalogue ainsi qu'observer plusieurs détails.
Il est également possible de trier la liste de vins en fonctions de certains critères.

Un utilisateur connecté(e) aura la possibilité d'ajouter,supprimer et modifier ses commentaires pour chaque vin
Il pourra également mettre une note et des données personnelles.

Étant le premier gros projet de l'année, nous avons pû apprendre beaucoup de choses.

Cela rajoute une corde á notre arc.
